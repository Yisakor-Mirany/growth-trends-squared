# Growth Trends Squared

## Objective
Square and sort an array of growth percentages.

## Approach
- Use two pointers (left and right)
- Compare squares of values
- Fill result array from largest to smallest

## Example
Input: [-5, -2, 0, 3, 10]  
Output: [0, 4, 9, 25, 100]

## Time Complexity
O(n)

## Space Complexity
O(n)

## Test Cases

### Normal Cases
- Mixed negative and positive values
- Sorted input arrays
- Multiple values

### Edge Cases
- Empty array
- Single element
- All negative values
